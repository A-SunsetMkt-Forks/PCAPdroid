<i>PCAPdroid</i> é um app de software livre que permite monitorar e exportar o tráfego de rede de seu dispositivo. O app simula uma VPN para conseguir uma captura não-root, mas ao contrário de uma VPN, o tráfego é processado localmente no dispositivo.

<b>Funcionalidades:</b>

* Registe e examine as conexões feitas pelo usuário e os apps de sistema
* Extraia SNI, consultas de DNS, URLs e endereços de IP remotos
* Crie regras para filtrar o bom tráfego e detectar facilmente as anomalias
* Extraia o tráfego para um arquivo PCAP, baixá-lo de um navegador ou enviá-lo para um receptor remoto para análise em tempo real (p.ex. wireshark)
* Use o app com o mitmproxy para descriptografar o tráfego de HTTPS/TLS (conhecimento técnico necessário)
* Em dispositivos com acesso root, você pode capturar os pacotes enquanto outros apps de VPN estão funcionando

Se você planeja usar o PCAPdroid para realizar análise de pacotes, verifique <a href='https://emanuele-f.github.io/PCAPdroid/quick_start#14-packet-analysis'>a seção específica</a> do manual.
